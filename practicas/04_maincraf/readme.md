

# Evidencia en Minecraft – Patrones de Diseño

## Tema
Patrón de Diseño: Singleton

## Descripción
Se construyó un mundo en Minecraft que representa el patrón de diseño Singleton,
el cual establece que solo puede existir una instancia de un objeto en el sistema.

La sala central simboliza la instancia única y el acceso controlado representa
la forma en que las demás clases interactúan con ella.

## Relación con la práctica en código
El patrón Singleton se implementa en programación para asegurar que solo exista
una instancia de una clase, por ejemplo en conexiones a bases de datos o
configuraciones globales del sistema.
